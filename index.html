Intro

Installable via npm install --save interface-transport, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save interface-transport

You can then require() interface-transport as normal:

const interfaceTransport = require('interface-transport')

In the Browser

Interface-transport should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of interface-transport, including index.js and index.min.js, are included in the /dist folder. Interface-transport can also be found on unpkg.com under

  • https://unpkg.com/interface-transport/dist/index.min.js
  • https://unpkg.com/interface-transport/dist/index.js
